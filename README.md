# Study Planner Pro Final

아이패드 11인치 가로 기준으로 맞춘 주간 학습 플래너입니다.

## 포함 기능

- 기본 시간표 자동 반영
- 대기 목록 → 시간표 드래그 배치
- 카드별 색상 / 장소 / 메모 수정
- 카드 안에서 개별 메모 표시
- 매일과제 별도 추가
- 매일과제 월~금 전체 반복 배치
- 매일과제 우선순위 정렬  
  - 국어 → 영어 → 수학 → 탐구
- 시험 요일 지정 / 해제
- 월·금 주간시험 토글
- 되돌리기 / 다시하기
- Safari localStorage 자동 저장

## 수정한 핵심 버그

- 대기 목록에서 드래그 시 복제처럼 남던 문제 수정
- 여러 카드가 한 시간에 있을 때 하나만 삭제해도 칸 전체가 지워지던 문제 수정
- 색상 선택 시 미리보기가 바로 반영되지 않던 문제 수정
- 카드 메모가 장소에 가려져 안 보이던 문제 수정
- 매일과제가 요일별 분산 배치되던 문제 수정
- 매일과제 중복 배치 누적 문제 수정

## 사용 방법

### 1. 빠른 추가
오른쪽 패널에서 이름, 장소, 색상, 유형을 입력하고 대기 목록에 넣습니다.

### 2. 드래그 배치
대기 목록 카드를 왼쪽 시간표 칸으로 드래그하면 이동합니다.

### 3. 카드 수정
카드를 눌러 이름 / 장소 / 메모 / 색상 / 유형을 수정할 수 있습니다.

### 4. 매일과제
매일과제를 추가한 뒤 `월~금 반복 배치`를 누르면,
매일과제 목록의 모든 항목이 월~금 각 요일에 반복 배치됩니다.
